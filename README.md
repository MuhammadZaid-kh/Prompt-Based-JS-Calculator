# 🧮 Prompt-Based JavaScript Calculator

A lightweight, interactive web-based calculator that performs basic arithmetic operations using JavaScript prompts and conditional logic.

## 🚀 Features
* **User-Friendly Prompts:** Simple pop-up inputs for numbers and operators.
* **Flexible Input:** Supports multiple symbols for operations (e.g., `x`, `X`, or `*` for multiplication).
* **Responsive Restart:** Features a dedicated "Restart Calculator" button to quickly refresh the page for new calculations.
* **Clean UI:** Styled with a sleek, rounded border and centered text using inline CSS.

## 🛠️ Tech Stack
* **HTML5:** For the basic structure and layout.
* **CSS3:** For styling the calculator container and the restart button.
* **JavaScript (Vanilla):** For handling the calculation logic and user input prompts.

## 📖 How to Use
1. Open the `index.html` file in any modern web browser.
2. Follow the prompt to **Enter the first digit**.
3. Follow the second prompt to **Enter the second digit**.
4. Type your desired operator from the list: `+`, `-`, `x`, `*`, `/`, `÷`, or `%`.
5. View the result directly on the web page.
6. Click the **"Restart Calculator"** button to start over.
