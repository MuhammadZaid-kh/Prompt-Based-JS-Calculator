# 🧮 Prompt-Based Calculator

Yeh mera pehla JavaScript project hai! Aik simple aur clean calculator jo browser prompts ke zariye input leta hai aur results display karta hai.

## 🚀 Features
* **Simple UI:** Clean box design with rounded borders.
* **Multiple Operators:** Supports Addition (+), Subtraction (-), Multiplication (x/*), Division (÷//), and Modulus (%).
* **Case Sensitivity:** Multiplication aur Division ke liye mukhtalif symbols (X, x, *) handle karta hai.
* **Restart Button:** Aik click se calculator ko dubara refresh kar ke use kar sakte hain.

## 🛠️ Tech Stack
* **HTML5** - Structure ke liye.
* **CSS3** - Styling aur layout ke liye.
* **JavaScript** - Logic aur calculations ke liye.

## 📖 How to Use
1. File ko kisi bhi browser mein open karen.
2. Pehla number enter karen.
3. Dusra number enter karen.
4. Diye gaye operators mein se koi aik type karen.
5. Screen par aap ko "Result" mil jaye ga!
